# Stock Pattern Scanner - CVNA Pattern Analysis

## Objective
Find stocks showing the same pattern as CVNA (Carvana) - massive collapse followed by massive rally - and identify the "first signal" to enter before the rally.

## The CVNA Pattern Framework

### Timeline
| Date | Price | Event |
|------|-------|-------|
| Aug 2021 | $370 | Peak (pandemic boom) |
| May 2022 | $80 | ADESA acquisition adds $9B debt |
| Dec 2022 | $3.55 | BOTTOM - 98% crash, bankruptcy fears |
| Mar 2023 | $5-8 | Debt restructuring completed |
| **June 8, 2023** | **$8 → $23** | **FIRST BIG SIGNAL: Surprise Q2 profitability** |
| Nov 2023 | $40+ | Q3 earnings beat |
| Feb 2024 | $50+ | First annual profit |
| Dec 2025 | $260+ | Added to S&P 500 |

### The 3 Key "First Signals"
1. **Debt Restructuring Completion** - Company renegotiated debt, extending maturities
2. **Cost-Cutting Results** - SG&A dropped, operating losses shrank
3. **Surprise Profitability Announcement** - Company guides profitability when everyone expects losses

### Pattern to Find Now
- Stock crashed 80%+ from highs
- Heavy debt but actively restructuring
- Improving operational metrics (costs down, margins improving)
- Has NOT yet announced profitability/turnaround
- High short interest (squeeze potential)
